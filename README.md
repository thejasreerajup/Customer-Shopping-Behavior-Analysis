
#  Customer Shopping Behavior Data Analyst Project

This project demonstrates a complete end-to-end data analytics workflow using customer shopping data. It shows how raw data can be transformed into meaningful business insights using Python, SQL, and Power BI.

---

## Project Overview

The objective of this project is to analyze customer purchasing behavior and generate insights that help in business decision-making.

The project includes:

- Data Cleaning & Exploratory Data Analysis (Python)
- Business Analysis using SQL
- Interactive Dashboard Creation (Power BI)
- Insight Generation and Reporting

---

## Business Objectives

- Analyze customer purchase patterns  
- Calculate revenue trends  
- Compare sales across gender and categories  
- Identify high-performing segments  
- Present insights using dashboards
